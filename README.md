# Менеджер документов (document-app)

JavaFX приложение для работы с документами (накладные, платёжки, заявки на оплату).  
Основные возможности:
- Создание/просмотр/удаление документов разных типов.
- Сохранение документов в H2 database.
- Экспорт/импорт документов в CSV.

Требования:
- Java 21
- Maven
- JavaFX 21

Запуск:
- mvn compile exec:java
